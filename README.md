# Directory Description
- `models/`: each saved model
- `photos/`: graphs and images from each dataset
- `cif100.ipynb`: model building and analysis for CIFAR-100 dataset
- `dataloaders.ipynb`: model building and analysis for vegetable image dataset
- `results.MD`: hyperparameter tuning results for each model in `cif100.ipynb`
